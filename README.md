# Design of Experiments (DOE) Tutorial
## Introduction

This course will cover the topic of design of experiments encompassing both ISYE 6413 - Design of Experiments and ISYE 7400 - Advanced Design of Experiments at the Georgia Institute of Technology broken down into Units *A1* to *A9* and *B1* to *B9* respectively.
## Course Overview
### Introduction to Design of Experiments

| Unit | Description                                                                     |
| ---- | ------------------------------------------------------------------------------- |
| A1   | [[Basic Concepts for Experimental Design and Introductory Regression Analysis]] |
| A2   | [[Experiments with a Single Factor - 1 Way ANOVA]]                              |
| A3   | [[Experiments with More than 1 Factor]]                                         |
| A4   | [[Full Factorial Experiments at 2 Levels]]                                      |
| A5   | [[Fractional Factorial Experiments at 2 Levels]]                                |
| A6   | [[Full Factorial and Fractional Factorial Experiments at 3 Levels]]             |
| A7   | [[Orthogonal Array Experiments]]                                                |
| A8   | [[Response Surface Methodology]]                                                |
| A9   | [[Robust Parameter Design]]                                                     |

### Advanced Design of Experiments

| Unit | Description                                                                   |
| ---- | ----------------------------------------------------------------------------- |
| B1   | Numerical Methods (Hilbert Spaces, Smoothing Splines, Kernel Methods)         |
| B2   | Statistical Methods (Kriging, GP Modeling, Cross-validation)                  |
| B3   | Experimental Designs (Latin hypercube designs, optimal designs)               |
| B4   | Sensitivity Analysis (Functional ANOVA decomposition, screening designs)      |
| B5   | Multi-fidelity Simulations (Autoregressive GPs, Nested space-filling designs) |
| B6   | Model Calibration (Non-linear regression, Bayesian calibration)               |
| B7   | Large-scale Experiments (Funcitonal response, large datasets)                 |
| B8   | Optimization (Expected Improvement Criterion, Robust Parameter Design)        |
| B9   | Uncertainty Quantification (Forward propagation, support points)              |
